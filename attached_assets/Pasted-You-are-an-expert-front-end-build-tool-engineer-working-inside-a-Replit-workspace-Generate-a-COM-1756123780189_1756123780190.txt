E. FUNCTIONAL EXTRAS

• SEO meta tags, OG defaults to `/static/logo.svg`.

────────────────────────────────────────
F. BUILD & NETLIFY DEPLOY
────────────────────────────────────────
Use **Eleventy 2.x**.

`package.json` scripts:
{
  "scripts": {
    "dev": "eleventy --serve --input=src --output=dist",
    "build": "eleventy --input=src --output=dist && node scripts/postbuild.cjs"
  },
  "devDependencies": {
    "@11ty/eleventy": "^2.0.0",
    "markdown-it": "^14.1.0"
  }
}

`eleventy.config.cjs`:
- Input: `src`
- Output: `dist`
- Pass-through copy: `static/` → `/`, `src/assets/` → `/assets/`
- Collections: posts sorted by date desc, pagination support
- Shortcodes/filters: reading time, date formatting, category color

`netlify.toml`:
[build]
  command = "npm run build"
  publish = "dist"
[[headers]]
  for = "/*"
  [headers.values]
    Cache-Control = "public, max-age=0, must-revalidate"
[[redirects]]
  from = "/*"
  to = "/404.html"
  status = 404

`/_redirects` (plain text file):
/*   /404.html   404

`/src/styles/global.css`:
- Define CSS variables from Brand Guidelines
- Base, typography, grid utilities, dark mode

`/src/scripts/main.js`:
- Mobile menu toggle
- Dark-mode toggle with `localStorage`
- Search (Fuse) populates results; debounce input
- Progressive enhancement only; no framework

`/src/layouts/base.njk`, `home.njk`, `post.njk`:
- Use partials for nav, hero, card, footer
- Ensure ALL asset href/src are **relative** (e.g., `./styles/global.css`, `./scripts/main.js`)

`/static/robots.txt`, `/static/favicon.ico`, `/static/logo.svg`, `/static/rss.xml` (generated at build), `/static/sitemap.xml` (generated at build)

`/README.md` (must include):
- Install: `npm i`
- Dev: `npm run dev`
- Build: `npm run build`
- **Netlify deploy**:
  1) Push to GitHub
  2) On Netlify: New site → Import from Git → Build command `npm run build`, Publish dir `dist`
  3) Add custom domain → enable HTTPS
  4) If path issues: verify assets are relative and rebuild

Finally, PRINT the file tree.
